# 💰 Loanify – Loan Approval Prediction

> 🚀 A machine learning system that predicts loan approval based on applicant data to support risk assessment and decision-making.

---

## 📌 Overview

Loanify is a **supervised machine learning** project that predicts whether a loan application should be approved.
It uses applicant information such as income, credit history, and employment status to evaluate financial risk.

---

## 🎯 Objectives

* Predict loan approval status
* Reduce financial risk
* Enable data-driven decisions

---

## 🧠 Key Concepts

* Supervised Learning
* Classification Algorithms
* Feature Engineering
* Model Evaluation

---

## ⚙️ Tech Stack

* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib

---

## ⚡ Workflow

1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature engineering
4. Model training (Logistic Regression, Decision Tree, etc.)
5. Evaluation (Accuracy, Precision, Recall)
6. Prediction

---

## 📊 Results

* Achieved reliable prediction performance
* Identified key factors affecting loan approval
